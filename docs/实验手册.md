# MambaVision-lite × IM-Fuse 实验手册

## 概述

本项目将 MambaVision 的混合 token mixing 策略引入 IM-Fuse 缺失模态脑肿瘤分割框架。核心改动仅替换每个模态分支的 Intra-modal Transformer 为 **HybridTokenEncoder** (MV-Mixer + Late Attention)，其余模块完全不变。

## 文件结构

```

├── mambavision_mixer.py      # [新] MambaVisionMixer3D + HybridBlock + HybridTokenEncoder
├── IMFuse_hybrid.py          # [新] 替换 IntraFormer 后的完整模型
```

## 实验版本

| 版本 | Token Encoder 结构 | 目的 | 优先级 |
|------|-------------------|------|--------|
| **Baseline** | Transformer × 1 (原始) | 基线 | 🔴 必做 |
| **E1: Mamba-only** | MV-Mixer × 2 | 验证纯 Mamba | 🟡 次要 |
| **E2: Hybrid-1** ⭐ | MV-Mixer × 1 + Transformer × 1 | **主方案** | 🔴 必做 |
| **E3: Hybrid-2** | MV-Mixer × 2 + Transformer × 1 | 边际收益 | 🟡 次要 |

控制方式: 通过 `--num_mamba_blocks` 和 `--num_attn_blocks` 参数切换：

| 版本 | `--num_mamba_blocks` | `--num_attn_blocks` |
|------|---------------------|---------------------|
| E1 | 2 | 0 |
| E2 ⭐ | 1 | 1 |
| E3 | 2 | 1 |

## 三阶段训练策略详解

### 学习率分组

| 阶段 | MV-Mixer | Hybrid Attn | Fusion | Conv Encoder | Decoder | Other |
|------|----------|-------------|--------|-------------|---------|-------|
| Stage 1 | 2e-4 | frozen | frozen | frozen | frozen | frozen |
| Stage 2 | 1e-4 | 5e-5 | frozen | frozen | frozen | frozen |
| Stage 3 | 5e-5 | 2e-5 | 1e-5 | 5e-6 | 1e-5 | 1e-5 |

每个阶段内使用独立的 Poly scheduler。阶段切换时重建 optimizer (清除 momentum)。

### Checkpoint 输出

每个阶段产生以下 checkpoint：

```
output/e2_hybrid1/
├── stage1_last.pth       # Stage 1 最后 epoch
├── stage1_best.pth       # Stage 1 最佳 val Dice
├── stage1_final.pth      # Stage 1 完成 (同 last)
├── stage2_last.pth
├── stage2_best.pth
├── stage2_final.pth      # → Stage 3 的输入
├── stage3_last.pth
├── stage3_best.pth       # ← 用于最终评估
└── stage3_final.pth
```

### 断点续训

如果某个阶段中断，可以从 `stageN_last.pth` 继续：

```bash
python train_hybrid.py \
    ... \
    --stage 2 \
    --stage_resume output/e2_hybrid1/stage2_last.pth
```

## 权重迁移说明

`load_imfuse_pretrained()` 函数处理 IM-Fuse → IMFuseHybrid 的权重映射：

| IM-Fuse 权重 | IMFuseHybrid 权重 | 处理 |
|-------------|------------------|------|
| `{mod}_encoder.*` | `{mod}_encoder.*` | 直接复制 |
| `{mod}_encode_conv.*` | `{mod}_encode_conv.*` | 直接复制 |
| `{mod}_decode_conv.*` | `{mod}_decode_conv.*` | 直接复制 |
| `{mod}_pos` | `{mod}_pos` | 直接复制 |
| `{mod}_transformer.cross_attention_list.0.fn.fn.qkv.*` | `{mod}_hybrid_encoder.blocks.{N}.mixer.qkv.*` | 映射到 Attn block |
| `{mod}_transformer.cross_attention_list.0.fn.fn.proj.*` | `{mod}_hybrid_encoder.blocks.{N}.mixer.proj.*` | 映射到 Attn block |
| `{mod}_transformer.cross_attention_list.0.fn.norm.*` | `{mod}_hybrid_encoder.blocks.{N}.norm1.*` | 映射到 Attn block |
| `{mod}_transformer.cross_ffn_list.0.fn.norm.*` | `{mod}_hybrid_encoder.blocks.{N}.norm2.*` | 映射到 Attn block |
| `{mod}_transformer.cross_ffn_list.0.fn.fn.net.{0,3}.*` | `{mod}_hybrid_encoder.blocks.{N}.mlp.{0,2}.*` | 仅维度匹配时复制 |
| — | `{mod}_hybrid_encoder.blocks.0.mixer.*` (MV-Mixer) | 随机初始化 |
| `mamba_fusion_layer*` | `mamba_fusion_layer*` | 直接复制 |
| `multimodal_transformer.*` | `multimodal_transformer.*` | 直接复制 |
| `decoder_*` | `decoder_*` | 直接复制 |

> **注意**: 原始 FFN hidden_dim=4096，新 MLP 的 hidden_dim 取决于 `hybrid_mlp_ratio`。
> 当 `hybrid_mlp_ratio=8` 时 hidden_dim=4096，FFN 权重可完全复用。
> 当 `hybrid_mlp_ratio=4` (默认) 时 hidden_dim=2048，FFN 权重维度不匹配，将随机初始化。

## 监控指标

### 训练关注

- **wandb 面板**: `stageN/loss`, `stageN/fusedice`, `stageN/val_Dice`
- **Stage 1 关键**: MV-Mixer 输出是否产生正面信号 (val 曲线)
- **Stage 切换**: 观察 loss 是否有 spike (应通过极低学习率缓解)

### 评估关注

- **全模态 (WT/TC/ET)**: 不应退化
- **T1c 缺失 → ET Dice**: 重点关注场景
- **单模态场景**: 最极端缺失下的表现

### 风险控制

1. **Stage 1 后无信号**: 暂停，检查 MV-Mixer 输出分布
2. **训练不稳定**: 降低 drop_path (0.05)，检查 gradient norm
3. **阶段切换 loss spike**: 已通过差异化 LR 缓解，可选加 5 epoch warmup
4. **显存超标**: 减小 batch_size 或降低 MLP ratio
